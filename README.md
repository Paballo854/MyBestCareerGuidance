# My Best Career Guidance Platform

A comprehensive Career Guidance and Employment Integration Web Application Platform for Lesotho.

## 🎯 Features
- Student course applications and career guidance
- Institution management and admissions  
- Company job postings and applicant matching
- Real-time notifications and application tracking

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js + Express.js  
- **Database**: Firebase Firestore
- **Authentication**: JWT
